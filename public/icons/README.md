# PWA Icons Needed

To complete the PWA setup, add these icon files to this directory:

## Required Icons:
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## How to Create:
1. Use your LoopLocal logo
2. Resize to 192x192 and 512x512 pixels
3. Save as PNG with transparency
4. Use tools like:
   - Canva (export at custom dimensions)
   - iloveimg.com/resize-image
   - Figma

## Tips:
- Use a square logo or add padding to make it square
- Make sure the logo is centered
- Use transparent background or solid color matching your brand
- The icons will be used when users install the app on their home screen
